Modigliani v1.0 - Documentación

Modigliani es una aplicación CRUD para la gestión de cuadros.

Arquitectura

Es una aplicación Web, cliente/servidor.

El frontend está basado en JavaScript (Documentación Frontend).

El backend está basado en PHP (Documentación Backend).

La base de datos utilizada es SQLite.

La comunicación entre frontend y backend se realiza mediante Ajax con un protocolo personalizado.
